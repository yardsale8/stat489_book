  1. In Robert McCloskey’s book Make Way for Ducklings, the names of the ducklings are Jack, Kack, Lack, Mack, Nack, Ouack, Pack, and Quack. This loop tries to output these names in order.

    prefixes = "JKLMNOPQ"
suffix = "ack"

words = [first + suffix for first in prefixes]

    Of course, that’s not quite right because Ouack and Quack are misspelled. Can you fix it?

  2. Create a function that takes a value n as input and constructs a multiplication table for whole numbers up to \(n\).

  3. Print out a neatly formatted multiplication table, up to 12 x 12. You should do this by constructing a string. For full credit, each column should be right-justified and your solution should include only comprehensions and lambda functions. Hint: Write a lambda function pads a number with the right number of spaces.

  4. Write a function that will return the number of digits in an integer.

  5. Write a function that reverses its string argument.

  6. Write a function that takes a string and mirrors its argument.

  7. Write a function that removes all occurrences of a given letter from a string.

  8. Write a function that recognizes palindromes. (Hint: use your reverse function to make this easy!). Hint: Use all, reversed and zip.

  9. Write a function that removes all occurrences of a string from another string.

  10. Create a list containing 100 random integers between 0 and 1000 (use iteration, append, and the random module). Write a function called average that will take the list as a parameter and return the average.

  11. Write a Python function that takes n and m as input and returns the maximum valu of a the list of n random integers between 0 and m. (Note:. there is a builtin function named max.)

  12. Write a function that computes the mean of n randomly selected values taken from a normal distribution with mean m and standard deviation sd.

  13. Write a function sum_of_squares(xs) that computes the sum of the squares of the numbers in the list xs. For example, sum_of_squares([2, 3, 4]) should return 4+9+16 which is 29:

  14. Write a function to count how many odd numbers are in a list.

  15. Sum up all the even numbers in a list.

  16. Sum up all the negative numbers in a list.

  17. Count how many words in a list have length 5.
